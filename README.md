# ミニノベルゲーム

【もうひとつ】というキーワードを探すインタラクティブなノベルゲームです。

## 使い方

`index.html` をブラウザで開くだけで、すぐにゲームをプレイできます。

- Node.jsやnpmのインストールは不要です
- インターネット接続が必要です（CDNからライブラリを読み込みます）
- GitHub Pagesでホスティングする場合、自動的に `index.html` が表示されます

## ゲームの遊び方

1. シナリオ選択画面で、プレイしたいシナリオを選択します
2. 選択肢を選んで物語を進めます
3. 【もうひとつ】というキーワードを含む選択肢を選ぶとスコアが加算されます
4. シナリオをクリアすると、選択画面に戻ります
5. すべてのシナリオで【もうひとつ】を獲得すると全クリアです！

## ファイル構成

- `index.html` - ゲーム本体（単一HTMLファイル）
