# 効果音ファイルの配置場所

## 「もうひとつ」ワードゲット効果音

複数の効果音ファイルからランダムに選んで再生されます。

**推奨ファイル名の例**:
- `word_get_01.wav`
- `word_get_02.wav`
- `word_get_03.wav`
- など（任意の数だけ追加可能）

## エンドクレジットBGM

エンドクレジット画面で流れるBGMです。

**ファイルパス**: `Assets/Audio/credits_bgm.wav`

**ファイル名**: `credits_bgm.wav`（任意の名前でも可）

## シナリオ選択画面BGM

シナリオ選択画面で流れるBGMです。シナリオに入ると一時停止し、戻ってきたら続きから再生されます。

**ファイルパス**: `Assets/Audio/selection_bgm.wav`

**ファイル名**: `selection_bgm.wav`（任意の名前でも可）

## Unityでの設定方法

### 効果音（「もうひとつ」ワードゲット）

1. 効果音ファイル（`.wav`）を `Assets/Audio/` フォルダに配置します
2. Unityエディタで `UIManagerUIToolkit` コンポーネントがアタッチされているGameObjectを選択します
3. Inspectorで `UIManagerUIToolkit` コンポーネントの `Audio` セクションを開きます
4. `Word Get Sounds` フィールドの **Size** を設定します（例: 3）
5. 各要素（Element 0, Element 1, Element 2...）に効果音ファイルをドラッグ&ドロップします
   - 任意の数の効果音を追加できます
   - 1つだけでも動作します

### エンドクレジットBGM

1. BGMファイル（`.wav`）を `Assets/Audio/` フォルダに配置します
2. Unityエディタで `UIManagerUIToolkit` コンポーネントがアタッチされているGameObjectを選択します
3. Inspectorで `UIManagerUIToolkit` コンポーネントの `Audio` セクションを開きます
4. `Credits BGM` フィールドにBGMファイルをドラッグ&ドロップします

### シナリオ選択画面BGM

1. BGMファイル（`.wav`）を `Assets/Audio/` フォルダに配置します
2. Unityエディタで `UIManagerUIToolkit` コンポーネントがアタッチされているGameObjectを選択します
3. Inspectorで `UIManagerUIToolkit` コンポーネントの `Audio` セクションを開きます
4. `Selection BGM` フィールドにBGMファイルをドラッグ&ドロップします

## 効果音が再生されるタイミング

- シナリオ画面で「もうひとつ」をクリックした時
- リザルト画面で「もうひとつ」をクリックした時（カウントダウン中）

**再生方法**: 設定された効果音の中からランダムに1つ選ばれて再生されます

## BGMが再生されるタイミング

### エンドクレジットBGM
- エンドクレジット画面を開いた時
- エンドクレジット画面を閉じた時に自動停止（2秒のフェードアウト）

**再生方法**: ループ再生されます

### シナリオ選択画面BGM
- シナリオ選択画面を開いた時（3秒のフェードイン）
- シナリオに入る時に一時停止（0.5秒のフェードアウト）
- シナリオ選択画面に戻ってきた時、一時停止した時刻から続きを再生（3秒のフェードイン）

**再生方法**: ループ再生されます。シナリオに入ると一時停止し、戻ってきたら続きから再生されます。

## 注意事項

- ファイル形式は `.wav` を推奨します（`.mp3` や `.ogg` も使用可能）
- ファイルサイズが大きすぎると、WebGLビルド時に問題が発生する可能性があります
- 効果音は短め（1-2秒程度）が推奨です
- 複数の効果音を設定することで、毎回異なる音が再生され、より自然な体験になります
- BGMは長めのファイルでも問題ありませんが、WebGLビルド時はファイルサイズに注意してください
